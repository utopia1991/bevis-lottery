# hanshan-lottery
## 特点
- 有点小意思
- 中国风
- 打开直接运行，暂时没有那么多花里胡哨的项目架构

## 运行
```bash
$ git clone https://github.com/hanxiansen/hanshan-lottery.git
$ cd hanshan-lottery
$ npm install
$ npm start
```
或者直接clone下来,双击打开index.html就可以了

## 抽奖
1. 运行项目
2. 一、二、三、纪念奖对应1、2、3、4按键，抽奖前选择对应的奖项
3. 点击开始抽奖按钮(或space空格键) —> 按钮显示可以抽奖了 —> 再点击抽奖按钮(或space空格键) —> 显示中奖人员
